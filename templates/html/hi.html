Module 1: The basics!

Introduction to HTML

HTML is the standard markup language used to create web pages. It provides the structure and content of a webpage. HTML consists of a series of elements, which are represented by tags. Tags are enclosed in angle brackets ("<>" and "") and are used to define the structure and formatting of the content.

                        
                                                
                            <!DOCTYPE html>
                            <html>
                            <head>
                                <title>My First Webpage</title>
                            </head>
                            <body>
                                <h1>Hello, World!</h1>
                                <p>This is a paragraph.</p>
                            </body>
                            </html>

Let's break down the different parts of this HTML template:

<!DOCTYPE html>: This declaration is used to specify the HTML version you are using. In this case, it's HTML5.
<html>: This is the root element of an HTML page. All other elements are contained within it.
<head>: This element contains meta-information about the document, such as the title of the page, links to external stylesheets, and scripts.
<title>: This element sets the title of the webpage, which appears in the browser's title bar or tab.
<body>: This is the container for the visible content of the webpage.
<h1>: This is a heading element. HTML has six levels of headings, from '<h1>' to '<h6>'. '<h1>' is the largest and '<h6>' is the smallest.
<p> This is a paragraph element. It's used to enclose blocks of text.
You can save the above code in an HTML file with a .html extension (e.g., index.html), and open it in a web browser to see the rendered result.

Introduction to HTML p/2

In addition to the basic structure, HTML provides a wide range of elements and attributes to structure and format content, create lists, add images, create links, and more. Here are a few examples:

                        
                                                
                            <!DOCTYPE html>
                            <html>
                            <head>
                                <title>My First Webpage</title>
                            </head>
                            <body>
                                <h1>Hello, World!</h1>
                                <p>This is a paragraph.</p>
                            </body>
                            </html>

Let's break down the different parts of this HTML template:

<img>: This element is used to embed an image in the webpage. It requires a src attribute to specify the path to the image file.
<a>: This element creates a hyperlink. The href attribute is used to specify the destination URL.
<ul> and <li>: These elements create an unordered list. <ul> is the unordered list container, and <li> represents each list item.
<table>, <tr>, <th>, and <td>: These elements are used to create tables. <table> represents the table itself, <tr> represents a table row, <th> represents a table header cell, and <td> represents a regular table cell.



This crash course provides only a glimpse of what HTML can do. To learn more, I recommend checking out online tutorials and resources such as the Mozilla Developer Network (MDN) HTML documentation (https://developer.mozilla.org/en-US/docs/Web/HTML) for detailed information on HTML elements, attributes, and best practices. Happy coding!
