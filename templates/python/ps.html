Module 1: The basics!

1. Basic syntax and data types

Python is an interpreted language. The print() function is used to display output to the screen. Python has several data types, including integers, floating-point numbers, strings, and Boolean values. The type() function is used to determine the type of a variable.

                                                
                            # Display output to the screen
                            print("Hello, world!")
                            
                            # Determine the type of a variable
                            x = 5
                            print(type(x)) # Output:

2. Variables and operators

Variables are used to store data. Variables can be assigned using the = operator. Arithmetic operators include +, -, *, /, % (modulus), and ** (exponentiation). Comparison operators include ==, !=, >, <, >=, and <=. Logical operators include and, or, and not.

                                                
                            # Assigning variables
                            x = 5
                            y = 3
                            
                            # Arithmetic operators
                            print(x + y) # Output: 8
                            print(x - y) # Output: 2
                            print(x * y) # Output: 15
                            print(x / y) # Output: 1.6666666666666667
                            print(x % y) # Output: 2
                            print(x ** y) # Output: 125
                            
                            # Comparison operators
                            print(x == y) # Output: False
                            print(x != y) # Output: True
                            print(x > y) # Output: True
                            print(x < y) # Output: False
                            print(x >= y) # Output: True
                            print(x <= y) # Output: False
                            
                            # Logical operators
                            print(x > 3 and y < 5) # Output: True
                            print(x > 10 or y < 1) # Output: False
                            print(not(x > 3 and y < 5)) # Output: False

3. Conditional statements (if-else)

The if statement is used to execute code if a certain condition is true. The else statement is used to execute code if the condition is false. The elif statement is used to execute code if multiple conditions need to be tested.

                                                
                            # Using if and else statements
                            x = 5
                            if x > 10:
                                print("x is greater than 10")
                            else:
                                print("x is less than or equal to 10")
                            
                            # Using elif statements
                            x = 3
                            if x > 5:
                                print("x is greater than 5")
                            elif x == 5:
                                print("x is equal to 5")
                            else:
                                print("x is less than 5")

4. Loops (for, while)

The for loop is used to iterate over a sequence (e.g., a list or string). The while loop is used to repeatedly execute code while a certain condition is true. The range() function can be used to generate a sequence of numbers to iterate over.

                                                
                            # Using a for loop
                            fruits = ["apple", "banana", "cherry"]
                            for fruit in fruits:
                                print(fruit)
                            
                            # Using a while loop
                            i = 0
                            while i < 5:
                                print(i)
                                i += 1
                            
                            # Using the range() function with a for loop
                            for i in range(5):
                                print(i)

5. Break and continue statements

The break statement is used to exit a loop. The continue statement is used to skip over an iteration and continue to the next one.

                                            
                        # Using a break statement
                        i = 0
                        while i < 5:
                            if i == 3:
                                break
                            print(i)
                            i += 1
                        
                        # Using a continue statement
                        i = 0
                        while i < 5:
                            i += 1
                            if i == 3:
                                continue
                            print(i)

6. Lists

A list is a collection of items, which can be of different types. Lists are mutable, which means that their contents can be changed. The append() method is used to add an item to the end of a list. The len() function is used to determine the length of a list.

                                            
                        # Creating a list
                        fruits = ["apple", "banana", "cherry"]
                        print(fruits)
                        
                        # Accessing list items
                        print(fruits[0]) # Output: "apple"
                        
                        # Changing list items
                        fruits[1] = "orange"
                        print(fruits) # Output: ["apple", "orange", "cherry"]
                        
                        # Adding an item to a list
                        fruits.append("pear")
                        print(fruits) # Output: ["apple", "orange", "cherry", "pear"]
                        
                        # Determining the length of a list
                        print(len(fruits)) # Output: 4

7. Dictionaries

A dictionary is a collection of key-value pairs, where each key is associated with a value. Dictionaries are mutable. The keys() method is used to return a list of the dictionary's keys. The values() method is used to return a list of the dictionary's values. 

                                            
                        # Creating a dictionary
                        person = {"name": "Alice", "age": 30, "city": "New York"}
                        print(person)
                        
                        # Accessing dictionary items
                        print(person["name"]) # Output: "Alice"
                        
                        # Changing dictionary items
                        person["age"] = 35
                        print(person) # Output: {"name": "Alice", "age": 35, "city": "New York"}
                        
                        # Adding an item to a dictionary
                        person["country"] = "USA"
                        print(person) # Output: {"name": "Alice", "age": 35, "city": "New York", "country": "USA"}
                        
                        # Returning a list of keys and values
                        print(person.keys()) # Output: ["name", "age", "city", "country"]
                        print(person.values()) # Output: ["Alice", 35, "New York", "USA"]

8. Functions

Functions are blocks of code that can be called multiple times with different arguments. Arguments are passed to a function using parentheses. Functions can return values using the return keyword. 

                                            
                        # Defining a function
                        def greet(name):
                            print("Hello, " + name + "!")
                            
                        # Calling a function
                        greet("Alice") # Output: "Hello, Alice!"

                        # Defining a function with a return value
                        def square(x):
                            return x ** 2

                        # Calling a function with a return value
                        result = square(3)
                        print(result) # Output: 9

9. File Handling

Files can be read and written using Python. The open() function is used to open a file. The read() method is used to read the contents of a file. The write() method is used to write to a file. The clode() method is used to close a file. 

                                            
                        # Writing to a file
                        file = open("example.txt", "w")
                        file.write("Hello, world!")
                        file.close()
                        
                        # Reading from a file
                        file = open("example.txt", "r")
                        content = file.read()
                        print(content) # Output: "Hello, world!"
                        file.close()

10. Regular expressions

Regular expressions are patterns that can be used to match strings. The re module is used to work with regular expressions in Python. The search() method is used to search for a pattern in a string. The findall() method is used to find all occurrences of a pattern in a string. 

                                            
                        import re

                        # Searching for a pattern
                        text = "The quick brown fox jumps over the lazy dog."
                        pattern = "fox"
                        result = re.search(pattern, text)
                        if result:
                            print("Match found!")
                        else:
                            print("Match not found.")
                        
                        # Finding all occurrences of a pattern
                        text = "The quick brown fox jumps over the lazy dog."
                        pattern = "the"
                        result = re.findall(pattern, text)
                        print(result) # Output: ["the", "the"]

11. Error handling

Errors can be handled using try-except blocks. The try block contains the code that might raise an error, and the except block contains the code that should be executed if an error occurs. 

                                            
                        # Handling an error
                        try:
                            x = 1 / 0
                        except ZeroDivisionError:
                            print("Division by zero!")

                        # Handling multiple errors
                        try:
                            x = 1 / 0
                        except ZeroDivisionError:
                            print("Division by zero!")
                        except:
                            print("An error occurred.")ò
